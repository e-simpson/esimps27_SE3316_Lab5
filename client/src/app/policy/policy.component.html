Admin procedure

If you recieve a claim follow the corresponding steps:

1. Using the admin account, find the image collection that is violating the DMCA policy. Usually the reporter should add this information in their report submission.

2. Once you have located the collection. Screen shot it and it's contents.

3. Record the collection creator, and the links of all images in the collection. Mainly the offending image.

4. Delete the offending collection.

5. Email the user that created the collection about what has happened.

6. Email evan.simp@gmail.com (site tech support) and NASA regarding this claim.
